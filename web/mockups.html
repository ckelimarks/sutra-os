Sutra UI Explorations

8 design directions for a single-orchestrator interface. Each mockup is a self-contained HTML file with mock data and interactions.

Round 1 — Initial Explorations
0
Sidebar + Chat
Original
Slack-inspired layout. Agent sidebar with status dots, central Sutra conversation, agent badges on responses. The starting point.
sidebar agent dots tooltip drill-down
A
Ambient OS
Spotlight meets HUD
Barely-there UI. Centered floating input like Spotlight. Responses fade after reading. Agent status as subtle edge-of-screen dots. The interface breathes.
minimal fading history breathing glow edge dots
B
Terminal Zen
Pure command line
Monospace everything. sutra > prompt. Colored left-borders per agent. Tab completion with @. CRT grid texture. Every character earns its place.
terminal monospace @tab-complete CRT aesthetic
C
Thread River
Flowing card stream
Cards flow down a central spine. Agent responses branch as tributaries with colored thread lines. Auto-collapse after 30s. Agent presence circles on the right edge.
cards auto-collapse agent circles hover highlight
Round 2 — Scale + Interaction
D
Orbital
Solar system dashboard
Agents orbit Sutra at the top. Active agents closer, idle agents drift outward. Animated dispatch lines. Collapsible to thin strip. 8 agents.
radial layout orbit animation file chips follow-up
E
Stream Columns
Bloomberg-style multi-stream
Sutra conversation center, agent activity streams flanking both sides. 10 agents across two rails. Most active closest to center. Expand cards inline.
3-column 10 agents reply targeting route animation
F
Depth Layers
Frosted glass depth
Sutra in front, agent grid behind at 40% opacity with blur. Double-click to pull an agent forward. Parallax on mouse move. See everything without switching.
backdrop-filter parallax z-depth 8 agents
G
Thread River Enhanced
Best of everything
Thread River core + inline agent replies, file chips with open/reveal, cloud layout toggle for 10+ agents, thread grouping, animated routing, status bar.
inline reply file chips cloud toggle thread groups 10 agents
Round 3 — Ultra Minimal, Full PRD Scope
H
Breath
Void + cursor. Nothing else.
Dark void. No chrome. You type on emptiness, Sutra responds, it fades. Tab reveals agents. All 10 PRD scenarios: dispatch, status, routing, files, approvals, replies, monitoring, memory, report zoom, budget alerts. Dieter Rams minimal.
zero chrome 10 scenarios keyboard-first fade history report zoom approvals
I
Pulse
One thought at a time
Each response replaces the previous — a living document. Edge pulse rings show agent activity as colored ripples. All 11 scenarios including Ollama local routing. Typography-only hierarchy. A sheet of paper as your OS.
single response edge pulses 11 scenarios report zoom approvals ollama
Round 4 — The Definitive Direction
K
River + Orbital + Attention Queue
Two-way relay, full PRD scope
Thread River conversation with orbital header and attention queue sidebar. Inbound agent completions flow into the stream. Priority-sorted queue (needs input > permission > completed > working). "While you were away" synthesis. 10 agents, 10 scenarios, file chips, inline replies, multi-computer support.
river + orbital attention queue inbound flow 10 scenarios file chips multi-machine
