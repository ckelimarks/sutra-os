Prototype Set · Agent Dispatch

Eighteen ways to dispatch agents & command an operation.

Nineteen frontend-only prototypes. Round 9 (Canvas, 19) is the latest — pure canvas with pinch/zoom/pan, no chrome except a 40px top bar, constellations spread in world space. Start at the top.

Round 09 (Opus) — Canvas · Pinch/zoom/pan · Minimal chrome
19
Canvas · Zoom · Pan · No chrome

Canvas

Strips all UI except a 40px glass top bar. The canvas IS the app. Pinch to zoom, drag to pan, double-click to zoom-to-fit an entity. Constellations spread in world space. Command strip slides up on selection. Mono-only. Night sky. Figma-style dot grid.

18
Constellation · Entity Forms · Night Sky

Constellation

Iterates on Bases (15). Each entity — LoveNotes, Prototypes, Content, Brandywine, BioSync, Contrarian — renders as its own signature constellation form (Hub / Branching / Stream / Pair / Seal) drawn from ATLAS + TAXONOMY. New nodes accrete into shape. Status drives brightness.

Dark

Round 06 (Opus) — Synthesis of all prior rounds

Desk · Synthesis · The refined bet

Desk

Warm editorial interface. Dispatch → agent arcs into a life-area lane. Tasks flow past NOW. Completed work grows a force-graph on each lane's right edge. Opus reviewed the prior 15 and made this.
Round 05 (Opus) — Bases as living knowledge graphs

Bases
